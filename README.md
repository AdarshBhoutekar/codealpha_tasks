# CodeAlpha Internship - Machine Learning Tasks

Welcome to my repository for the **CodeAlpha Machine Learning Internship**. This repository contains the projects and tasks I completed during my internship, showcasing practical applications of machine learning techniques ranging from credit scoring to computer vision and disease prediction.

##  Project Overview

### Task 1: Credit Wise (Loan Approval Prediction)
**Description:**  
A machine learning project designed to automate the loan approval process. The model analyzes applicant details to predict creditworthiness and loan eligibility, helping financial institutions mitigate risk.
- **File:** `Task 1/credit_wise_project 1.ipynb`
- **Key Techniques:** Data Preprocessing, Classification Algorithms, Model Evaluation.

---

### Task 3: Handwriting Recognition (MNIST & EMNIST)
**Description:**  
This project focuses on building a Convolutional Neural Network (CNN) to recognize handwritten digits and letters. It utilizes the famous MNIST dataset and extends capabilities with the EMNIST dataset for character recognition.
- **File:** `Task 3/Task 3.ipynb`
- **Datasets:**
    - [MNIST in CSV](https://www.kaggle.com/datasets/oddrationale/mnist-in-csv)
    - [EMNIST (Extended MNIST)](https://www.kaggle.com/datasets/crawford/emnist)
- **Key Techniques:** Deep Learning, CNN Architecture, Image Processing.

---

### Task 4: Disease Prediction (Cardiovascular Disease)
**Description:**  
A predictive model built to estimate the risk of cardiovascular disease in patients. By analyzing various health metrics, the model assists in early diagnosis and health monitoring.
- **File:** `Task 4/Task 4 ML.ipynb`
- **Key Techniques:** Feature Engineering, Logistic Regression/Random Forest, Health Data Analysis.
